# Security Policy

## Supported Versions

Currently, only the latest version of `qti-reporter` is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please report it by opening a GitHub Issue with the "security" label or contact the maintainers directly.
